# Claude Code Skills

Personal collection of [Claude Code](https://claude.com/claude-code) skills.

## Install

Copy a skill folder into your skills directory, then invoke it in Claude Code:

```bash
cp -r premium-design-loop ~/.claude/skills/
# then: /premium-design-loop
```

## Skills

| Skill | What it does |
|-------|--------------|
| **premium-design-loop** | Iteratively evaluate and upgrade a webpage's visual craft against premium design standards — typography, hierarchy, colour, light/motion (光影), mobile, a11y, finishing. Each round: render → score → optimize the weak dimensions → re-score, with before/after screenshots. Built for "make it look more premium / 更高级". |
